# Regla 017 – No aumentar carga si la fatiga es alta

## Objetivo

Evitar que el sistema aumente la carga de entrenamiento cuando el atleta presenta altos niveles de fatiga.

## Descripción

Antes de completar volumen con sesiones Z2 o extender sesiones existentes, el motor debe evaluar el nivel de fatiga actual del atleta.

## Escala

- 0 a 3: Fatiga baja
- 4 a 6: Fatiga moderada
- 7 a 10: Fatiga alta

## Comportamiento

### Fatiga baja (0-3)

- Planificación normal.
- Se permite completar volumen disponible.
- Se permite extender sesiones.

### Fatiga moderada (4-6)

- Reducir expansión de volumen.
- Limitar sesiones complementarias.

### Fatiga alta (7-10)

- No agregar sesiones Z2.
- No extender sesiones.
- Mantener únicamente sesiones esenciales.

## Beneficio

Prioriza la recuperación y reduce el riesgo de sobreentrenamiento.
